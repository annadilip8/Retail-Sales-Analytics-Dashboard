# Retail Sales Analytics & Business Intelligence Dashboard

## Project Overview
This project performs end-to-end data analysis on a retail sales dataset using Python, SQL, and Power BI.

## Key Features
- Data Cleaning and Exploratory Data Analysis (Pandas)
- SQL Queries using GROUP BY and Aggregations
- KPI Reporting
- DAX Measures (Distinct Count)
- Sales Forecasting using Linear Regression
- Interactive Power BI Dashboard

## KPIs Built
- Total Sales, Orders, Customers
- Total Sales by Region
- Total Sales By Year
- Top 10 Customers

## Tech Stack
Python | SQL | Power BI | AI/ML
